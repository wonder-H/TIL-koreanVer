> 데이터의 불변성

- 자바스크립트의 원시 타입은 메모리 영역에서 값을 변경할 수 없다(메모리 주소변경만 가능)
- 객체 / 함수 / 배열은 참조 형태로 되어있어 직접적인 값 변경 가능

- 불변성은 데이터 선언 후 값의 변형 없이 원본을 유지하기 위한 것
- 때문에 불변성을 위해 데이터 수정 후에는 수정본을 새로운 데이터 값으로 다시 만든다
